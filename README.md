# Carolina Prado — Academic Portfolio

Personal academic website hosted via GitHub Pages.

## Structure

```
.
├── index.html          # Main site file
├── slides.pdf          # Presentation slides
├── cv.pdf              # Curriculum Vitae
├── assets/
│   ├── photo.png       # Profile photo
│   ├── capa.jpeg       # Book cover
│   ├── contracapa.jpeg # Book back cover
│   ├── hist1.jpeg      # Book interior page
│   ├── hist2.jpeg      # Book interior page
│   └── hist3.jpeg      # Book interior page
```

## How to update

### Update content
Edit `index.html` directly. The main sections are clearly marked with HTML comments:
- `<!-- About -->` — bio, research interests, email and GitHub links
- `<!-- Research -->` — papers, abstracts, presentations
- `<!-- Teaching -->`
- `<!-- Books -->`

### Add a new paper
Find the `<!-- Research -->` section in `index.html` and copy the paper block, updating the title, abstract, draft link, slides link, and presentations.

### Update slides
Replace `slides.pdf` in the root of the repository with your updated file.

### Update the CV
Replace `cv.pdf` in the root of the repository with your updated file.

### Update the profile photo
Replace `assets/photo.png` with a new PNG file.
